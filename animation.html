编剧
性质：全职
  岗位职责 :
  • 1、根据故事创意提交故事梗概；
  • 2、创建和设定故事角色的性格和故事的人文背景；
  • 3、参加剧本策划和评审会议，积极配合剧本修改工作；
  • 4、整理提纲，设计戏剧性情节，并撰写剧本和编写脚本；
  • 5、具备出色创造力，较强的文字功底和动漫漫画镜头语言，擅长故事编写，思维活跃、文字幽默、作品故事生动，能驾驭长短篇不同类型的故事编创。
  任职要求 :
  • 1、有写动画或电影剧本创作经验一年以上者（在创意创作方面有特殊才能或经历者，可放宽学历要求）；
  • 2、有较强剧本创作能力和一定的市场判断力，了解青少年流行语言与文化，可独立完成动漫剧本创作，有对中国传统文化爱好者优先考虑；
  • 3、具有较强的学习能力，具备高效高质完成任务的能力，责任心强，具有良好的沟通能力和团队合作精神者。可以适应高强度工作；
  • 4、热爱互联网，熟悉互联网用户和粉丝思维；
  • 5、有剧本编写经验或儿童故事作品发表者优先；
品牌市场总监
性质：全职
  岗位职责 :
  • 1、分析品牌定位、价值、特点，带领策划团队深入研究客户需求，完成各类策划案和推广方案和品牌策划指标；
  • 2、搭建公司市场营销体系，制定市场渠道的开发策略，并推动市场渠道合作落地；
  • 3、带领策划团队，协同相关团队共同完成品牌策划指标；
  • 4、参与并执行公司制定的夹断发展目标，针对品牌推广宣传和商业开发目标做好执行策略的制定；协同相关部门推进品牌运营。
  任职要求 :
  • 1、本科以上学历，动画、广告、新闻、媒体传播等相关专业；
  • 2、文字功底深厚，并具有超凡的创意能力，擅长软文、品牌推广、商业开发、活动策划案的撰写；
  • 3、拥有很强的策划能力，有成功策划案例，熟悉品牌、广告］营销等策划及文案；
  • 4、熟悉动漫产业或文化传媒产业，了解互联网行业，熟悉互联网和新媒体的传播渠道；
  • 5、逻辑思维能力和创意能力强，善于沟通、协调，且具有强烈的责任感和事业心；
  • 6、有4A经验或其他互联网公司相关经验优先考虑。
授权经理

  • 4、对所辖授权产品／进行市场调研、数据分析与形象维护，与现有／潜在厂商的会面、产品／形象的展示、品牌管理、合同／订单的接收与跟进、应收账款的催收，以及公司客户的售后服务等工作；
  任职要求 :
  • 1、专科或以上学历；三年以上工作经历。
  • 2、有动画形象和相关衍生产品授权经验者，有渠道资源者优先。
  • 3、具备创业魄力与创业者心态，极富工作激情，同时有较强的抗压能力；
  • 4、具备优秀的销售特质（强烈的成就欲望、目的性强、自信、优秀的沟通影响力、行动力强），对客户需求有敏锐的感受和判断，头脑灵活；
  • 5、具备品牌建设及品牌管理等相关知识，对品牌授权领域有一定的了解，对国内外动漫产生及卡通形象有一定了解；
  • 6、有本土公司的工作经验,了解国内市场，适应力强，有独立负责某项业务整体运营极管理经验者优先；
  • 7、综合素质佳，优秀的学习能力，具有敏锐的市场洞察力及反应力；
  • 8、为人正直、具备优秀的职业道德，思路开阔，创造性解决问题能力强。

  • 8、能根据公司要求调节出高质量动作表演，高效地按时完成任务；
  • 9、有三维或二维部门生产流程管理从业经验尤佳；
  • 10、主动，负责，积极，善于沟通，有较强的团队协作能力，能接受因项目需要的加班。

新媒体运营
性质：全职
  岗位职责 :
  • 1、负责整个社群的规划以及执行，整体负责社群内容，社群氛围的营造，打造高粘性的社群；

  • 1、熟悉社区运营、粉丝建设、SNS推广工作经验，熟悉各大互联网社区文化，有丰富SNS合作资源优先；
